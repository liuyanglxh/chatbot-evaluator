## 📊 测试数据概览

**总计**: 15条测试数据
**命名规则**: `Correctness-X分-描述（对比点）`
**覆盖**: 1-5分的评分标准，每个分数3条数据
**领域**: 保险业务场景
**更新时间**: 2025-01-23

---

## 🎯 命名规范

### 命名格式

```
Correctness-X分-描述（对比点）
```

**组成部分**:
- **Correctness**: 评估指标类型前缀
- **X分**: 评分等级（1-5分）
- **描述**: 测试点的简要说明
- **（对比点）**: 具体的差异说明

### 示例

```
Correctness-1分-主题完全不同（保险vs旅游）
Correctness-2分-数值不一致（25vs26元）
Correctness-3分-流程细节不足（3步vs1步）
Correctness-4分-同义词替换（费用vs价格）
Correctness-5分-完全一致（保险退费流程）
```

## 🎯 命名优势

### 1. 清晰的指标分类
```
Correctness-1分-主题完全不同
Faithfulness-2分-细节不符
AnswerRelevancy-3分-部分相关
```
一眼就能看出数据属于哪个评估指标

### 2. 便于管理
- 按指标类型排序
- 按评分等级筛选
- 快速定位特定数据

### 3. 避免冲突
不同指标的测试数据可以有相同的后缀描述，不会产生命名冲突

## 🚀 使用建议

### 添加新数据时

**命名格式**:
```python
test_data = {
    "name": "Correctness-X分-描述（对比点）",
    "question": "问题内容",
    "answer": "实际答案",
    "context": "期望答案"
}
```

**示例**:
```python
test_data = {
    "name": "Correctness-2分-时间不一致（10点vs11点）",
    "question": "客服热线的工作时间是什么？",
    "answer": "客服热线的工作时间是上午9点到下午11点。",
    "context": "客服热线的工作时间是上午9点到下午10点。"
}
```

### 管理测试数据

**按指标筛选**:
```python
# 获取所有Correctness指标的数据
correctness_data = [d for d in test_data_list if d['name'].startswith('Correctness-')]
```

**按分数筛选**:
```python
# 获取所有1分数据
score_1_data = [d for d in test_data_list if '1分-' in d['name']]
```
